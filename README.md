# Portfolio Website

This is a personal portfolio website built using HTML and CSS.

The project was created by following a tutorial as part of my learning process. It helped me understand how real portfolio websites are structured and how different sections come together in a complete layout.

---

## Tech Stack
- HTML
- CSS

---

## Features
- Hero section with introduction
- About section
- Skills section
- Projects layout
- Contact section (basic structure)
- Responsive layout basics

---

## Purpose of the Project
This project was built to:
- Understand how a complete website is structured
- Learn how to design multi-section layouts
- Practice HTML and CSS in a real-world project format
- Gain confidence in building websites from scratch using references/tutorials

---

## What I Learned
- Structuring a multi-section webpage
- CSS Flexbox and layout alignment
- Spacing, positioning, and responsive design basics
- How real portfolio websites are organized
- Converting a design/tutorial into actual code

This project was built by following a tutorial for learning purposes and to strengthen my understanding of frontend development.
